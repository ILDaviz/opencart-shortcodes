# OpenCart Shortcodes 

OpenCart Shortcodes is a library and set of modification to enhance OpenCart with shortcodes based on Wordpress API.

## Installation

1. Download and extract OpenCart Shortcodes.
2. Extract the zip package and use admin installer to upload shortcodes.ocmod.zip
3. Installation is done

## Quick Test
1. Copy and Paste content quick-sample-code.txt to Extension > Information content
2. Open on front site to check if Shortcodes working
