link_product

Return product link with or without the category and manufacture parent.

Format

Self-closing shortcode will return link with product name, while enclosing return link with custom text.

Self-closing

[link_product id="xx" path="xx_xx" brand="xx" ssl="1" title="lorem ipsum" /]

Enclosing

[link_product id="xx" path="xx_xx"]custom text[/link_product]

Options

Attribute Description.
id Product id you want to link.
path Category id in single or with parent. Single category: 14, with parent 7_14.
brand Manufacturer id of product. If contain "path" and "brand", the brand will be abandoned.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

link_custom

Return link based on your input. This shortcode similiar to OpenCart API:

 $this->url->link($route, $args, $ssl)

Format

Enclosing: return link with custom text.

[link_custom route="information/information" args="information_id=3"]custom text[/link_custom]

Options

Attribute Description.
route Route page.
args Arguments of the route.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

link_store

Return link based on your input across multi-store. This shortcode similiar to link_custom shortcode, but pointing to other store (related to OpenCart multistore feature).

Format

Enclosing: return link with custom text.

[link_store store="x" route="information/information" args="information_id=3"]custom text[/link_store]

Options

Attribute Description.
store Store id.
route Route page.
args Arguments of the route.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

module_product

Return product module: featured, latest, bestseller or special module.

Format

Self-closing: return the module based on type.

[module_product type="featured" limit="5" img_w="100" img_h="100" /]

Options

Attribute Description.
type Module type to show. Options: featured, latest, bestseller, special
limit Product limit to show. Default: 5.
img_w Product image width. Default: 80
img_h Product image height. Default: 80
