link_product

Return product link with or without the category and manufacture parent.

Format

Self-closing: return product name with link.

[link_product id="xx" path="xx_xx" brand="xx" ssl="1" title="lorem ipsum" /]

Enclosing: return custom text with link.

[link_product id="xx" path="xx_xx"]custom text[/link_product]

Options

Attribute Description.
id Product id from admin > catalog > products.
path Category id in single or with parent. Single category: 14, with parent 7_14.
brand Manufacturer id of product. If contain "path" and "brand", the brand will be abandoned.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

link_custom

Return link based on your input. This shortcode similiar to OpenCart API:

 $this->url->link($route, $args, $ssl)

Format

Enclosing: return custom text with link.

[link_custom route="information/information" args="information_id=3"]custom text[/link_custom]

Options

Attribute Description.
route Route page.
args Arguments of the route.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

link_store

Return link based on your input across multi-store. This shortcode similiar to link_custom shortcode, but pointing to other store (related to OpenCart multistore feature).

Format

Enclosing: return custom text with link.

[link_store store="x" route="information/information" args="information_id=3"]custom text[/link_store]

Options

Attribute Description.
store Store id.
route Route page.
args Arguments of the route.
ssl Format output with http or https. Options: 1 or 0. Default: 0.
title Title of the link.

module_product

Return product module: featured, latest, bestseller or special module.

Format

Self-closing: return module based on type.

[module_product type="featured" limit="5" img_w="100" img_h="100" /]

Options

Attribute Description.
type Module type to show. Options: featured, latest, bestseller, special
limit Product limit to show. Default: 5.
img_w Product image width. Default: 80
img_h Product image height. Default: 80

module_slideshow

Return slideshow module.

Format

Self-closing: return module slideshow.

[module_slideshow id="x" limit="5" img_w="100" img_h="100" /]

Options

Attribute Description.
id Banner id from admin > system > design > banners
limit Product limit to show. Default: 5.
img_w Product image width. Default: 80
img_h Product image height. Default: 80

login

User required to login to read the rest of the content.

Format

Enclosing: hide content from unlogged user, and show info to login.

[login]content[/login]
[login message='Silahkan <a href="%s">login</a> untuk melihat halaman ini.']content[/login]

Options

Attribute Description.
message Message to inform users that login is required to read the rest of content.
Default: Please <a href="%s">login</a> to read the rest of the post.
suffix Stylesheet class suffix.

video

Embed video from YouTube or Vimeo.

Format

Self-closing: return video.

[video type="vimeo" id="xxx" vid_w="450" vid_h="280" /]

Options

Attribute Description.
type Video source, options: youtube, vimeo. Default: youtube
id Video identifier.
vid_w Video width. Default: 450
vid_h Video height. Default: 280
autoplay Autoplay the video? Options: 1 or 0Default: 0

image

Embed image with cache option.

Format

Self-closing: return image.

[image src="image/data/xxx.jpg" img_w="450" img_h="280" title="" alt="" align="" /]

Options

Attribute Description.
src Image source.
img_w Image width. Default: 200
img_h Image height. Default: 200
title Image title show when user hover.
alt Alternative text when image not loaded.
align Align the image. Options: left, right, center
