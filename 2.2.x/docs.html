Installation

Extension Installer

  1. Uploading

    Go to Extensions > Extension Installer and upload file installer/shortcodes.ocmod.zip.

    NOTE If you previously have Shortcodes installed, you will see replace file list. Proceed to replace them.

    NOTE If you get warning "using the same ID code as the one you are trying to upload!" mean you installing same version. Click the clear button and no need to proceed.

  2. Installation

    Go to Extensions > Modifications, refresh modification to generate new modification cache.

    NOTE If you see modification with same name, delete the one with lower version before refresh modification.

  3. Quick Test

    1. Copy and Paste content of file quick-sample-code.txt to Extension > Information content. You san use "Shortcodes sample" as title.
    2. Open on front site to check if Shortcodes working

Manual Install

If you have issues using Extension Installer, proceed with this manual installation.

  1. Upload Files

    Use FTP software and upload folders inside manual/upload to your site root.

    NOTE Do not upload the folder upload/, but folders inside it.

  2. Upload OcMod

    Go to Extensions > Extension Installer and upload file manual/shortcodes.ocmod.xml.

    NOTE If you get warning "using the same ID code as the one you are trying to upload!" mean you installing same version. Click the clear button and no need to proceed.

  3. Installation

    Next installation step is the same as Extension Installer: Installation.
